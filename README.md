# ✨ DevTechW7 Dashboard ✨

**DevTechW7 Dashboard** é uma aplicação front-end construída com **Vue 3**, **Vuetify** e configurada como um **PWA (Progressive Web App)**. Esta aplicação serve como painel administrativo para o backend DevTechW7, oferecendo uma interface amigável e responsiva para gerenciar diversos módulos e realizar operações de CRUD em entidades. A aplicação é hospedada em um ambiente **Docker** e é servida por um backend básico em **Node.js com Express**.

---

## 🚀 Tecnologias Utilizadas

- **Vue 3**: Framework JavaScript progressivo para construir interfaces de usuário.
- **Vuetify**: Biblioteca de componentes baseada no Material Design para Vue.
- **PWA**: Implementação de Progressive Web App para possibilitar uma experiência de aplicativo nativo.
- **Node.js & Express**: Servidor básico para servir a aplicação Vue em produção.
- **Docker**: Empacotamento da aplicação para garantir consistência e facilidade de deploy.

---

## 🌟 Funcionalidades

- **Autenticação JWT**: Acesso seguro para usuários autenticados.
- **Gestão de Módulos**: Exibição de cards com diferentes módulos e aplicativos gerenciáveis.
- **CRUD Completo**: Operações de criação, leitura, atualização e exclusão para entidades como.
- **Interface Responsiva**: Design otimizado para dispositivos móveis e desktops.
- **Experiência PWA**: Possibilidade de instalar a aplicação como um app em dispositivos móveis e desktops.
